This folder is meant for content and configuration you update frequently.

What to edit:
- edit/data/people.yml: People page members and sections.
- edit/data/service.yml: Service activities and images.
- edit/data/projects_page.yml: Projects page sections and labels.
- edit/data/icl.yml: Integrated Computing Lab sections and description.
- edit/data/repositories.yml: GitHub users and repositories list.
- edit/data/socials.yml: Social media links and contact info.
- edit/data/cv.yml: CV data (if not using JSON format).

Publications:
- Publications are stored in _bibliography/papers.bib (BibTeX format).
- To add a new publication, add a BibTeX entry to that file.
- Optional fields: pdf, preview, abstract, code, website, selected.
- Put PDF files in assets/pdf/ and preview images in assets/img/publication_preview/.

Notes:
- These files are used by Jekyll via _config.yml (data_dir).
